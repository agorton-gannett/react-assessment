# Frontend stack

## How to use

Download the example [or clone the repo](https://github.com/mui-org/material-ui):

```sh
git clone https://github.com/vercel/next.js/issues/1791
cd gannett-frontend-assessment
```

Install it and run:

```sh
npm install
npm run dev
```
